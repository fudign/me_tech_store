# Xiaomi Store - Интернет-магазин техники

## What This Is

Интернет-магазин техники Xiaomi для продажи смартфонов и разнообразной техники бренда Xiaomi (умный дом, ноутбуки, аудио, телевизоры и аксессуары). Покупатели могут просматривать каталог, фильтровать товары и оформлять заказы без регистрации. Администратор управляет всем контентом сайта через удобную панель управления, похожую на OpenCart.

## Core Value

Покупатели могут легко найти нужную технику Xiaomi, увидеть все характеристики и цены, и быстро оформить заказ без лишних шагов. Администратор может полностью контролировать весь контент сайта и обрабатывать заказы.

### Out of Scope

- Регистрация и личные кабинеты пользователей — пока не нужно, гостевой checkout достаточен
- Email уведомления покупателям — сначала фокус на уведомления админу
- Telegram уведомления — может быть добавлено позже
- Многоязычность — пока только русский язык
- Мобильное приложение — только веб-версия
- Система отзывов и рейтингов — не в первой версии
- Программа лояльности и бонусы — отложено

## Context

**Существующие активы:**
- Готовый HTML дизайн сайта (generated-page.html) с использованием Tailwind CSS
- Дизайн включает: главную страницу, каталог категорий, карточки товаров, корзину
- Визуальный стиль: современный, минималистичный, в оранжевых тонах Xiaomi

**Целевая аудитория:**
- Покупатели из Кыргызстана (Бишкек)
- Ищут технику Xiaomi с официальной гарантией
- Хотят быстро оформить заказ без сложной регистрации

**Бизнес-модель:**
- Продажа техники Xiaomi с наценкой
- Официальная гарантия 1 год
- Бесплатная доставка при заказе от $50
- Рассрочка через банки-партнеры

**Вдохновение:**
- OpenCart — администратор знаком с этой системой и хочет похожий уровень контроля
- Современные интернет-магазины техники — чистый дизайн, удобная навигация

## Constraints

- **Tech stack**: PHP + MySQL — администратор хочет использовать эти технологии
- **Existing design**: HTML дизайн уже готов и должен быть интегрирован без значительных изменений
- **SEO**: Обязательная оптимизация под Google и Яндекс — мета-теги, ЧПУ URL, sitemap
- **Admin UX**: Админка должна быть понятной, как в OpenCart — администратор уже знаком с таким интерфейсом
- **No registration**: Покупатели заказывают без регистрации — быстрый checkout критичен
- **Deployment**: Сначала локальная разработка, потом перенос на хостинг
- **Language**: Только русский язык в первой версии
- **Payment integration**: Нужна интеграция с платежной системой для онлайн оплаты
